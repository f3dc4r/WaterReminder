# 💧 Water Reminder

A JetBrains IDE plugin that reminds you to drink water while coding.

## 📖 Read the story
I wrote an article about how I built this plugin and ported it to VS Code:
[I built a water reminder plugin for JetBrains and VS Code — here's how and why](https://dev.to/f3dc4r/i-built-a-water-reminder-plugin-for-jetbrains-and-vs-code-heres-how-and-why-1ac9)

## Features
- 💧 Water drop icon in the status bar (bottom right)
- Configurable reminder interval (1–480 minutes)
- Enable/disable with one click
- Click the icon to open settings

## Installation
Install directly from the [JetBrains Marketplace](https://plugins.jetbrains.com).

## Usage
1. After installation, a 💧 icon appears in the bottom right status bar
2. Click the icon to open settings
3. Set your preferred reminder interval
4. Stay hydrated! 🚀
